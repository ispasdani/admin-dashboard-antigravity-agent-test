# Brand Guidelines

## Brand Name

Hirezy

## Brand Essence

Hirezy is a modern, data-driven hiring platform designed to make recruitment feel simple, human, and efficient. The brand balances professionalism with approachability, using clean layouts, soft colors, and clear data visualization.

---

## Brand Personality

- **Modern & Clean** – Minimal UI, generous whitespace, and soft contrasts
- **Trustworthy** – Calm colors, structured layouts, and consistent components
- **Human-Centered** – Friendly shapes, readable typography, and intuitive navigation
- **Data-Informed** – Clear metrics, charts, and dashboards without visual overload

---

## Logo Usage

- The logo should appear in a **simple, flat style** with no heavy shadows or gradients.
- Maintain clear space around the logo equal to the height of the logo icon.
- Use the logo primarily on light or neutral backgrounds.

---

## Color Palette

### Primary Colors

| Color Role       | Hex       | Usage                                 |
| ---------------- | --------- | ------------------------------------- |
| Primary Accent   | `#C9F24D` | Highlights, key metrics, CTAs         |
| Secondary Accent | `#BFD1FF` | Cards, secondary panels               |
| Brand Blue       | `#7F8CFF` | Charts, icons, informational elements |

### Neutral Colors

| Color            | Hex       | Usage                 |
| ---------------- | --------- | --------------------- |
| Background       | `#F7F8FA` | App background        |
| Card Background  | `#FFFFFF` | Cards, tables, panels |
| Border / Divider | `#E6E8EC` | Lines, separators     |
| Text Primary     | `#1F2937` | Headings, main text   |
| Text Secondary   | `#6B7280` | Labels, helper text   |

### Status Colors

| Status  | Color     | Usage                              |
| ------- | --------- | ---------------------------------- |
| Success | `#A3E635` | Growth indicators, positive trends |
| Warning | `#FACC15` | Pending or attention-needed states |
| Error   | `#F87171` | Declines, negative metrics         |

---

## Typography

### Primary Font

**Inter** (or similar modern sans-serif)

### Font Weights

- **Regular (400)** – Body text
- **Medium (500)** – Labels, navigation
- **Semibold (600)** – Card titles
- **Bold (700)** – Key metrics, headlines

### Type Scale (Suggested)

| Usage          | Size    |
| -------------- | ------- |
| Page Title     | 24–28px |
| Section Header | 18–20px |
| Card Title     | 14–16px |
| Body Text      | 13–14px |
| Caption / Meta | 12px    |

---

## Layout & Spacing

- Use an **8px spacing system** (8, 16, 24, 32).
- Cards should have:
  - Padding: `16–24px`
  - Border radius: `12px`
  - Soft shadow or subtle border
- Dashboards should prioritize **scanability** over density.

---

## Components

### Cards

- Rounded corners
- Light shadow or border
- Clear hierarchy: title → value → context

### Buttons

- Primary: Bright accent color with dark text
- Secondary: Neutral background with subtle border
- Icon buttons should be circular or softly rounded

### Tables

- Minimal borders
- Zebra striping optional
- Clear column headers with muted color

---

## Icons & Illustrations

- Use **outline or duotone icons**
- Avoid overly detailed or skeuomorphic styles
- Icons should feel friendly and consistent in stroke width

---

## Charts & Data Visualization

- Use soft, desaturated colors
- Avoid harsh contrasts
- Prefer:
  - Donut charts
  - Bar charts
  - Simple line charts
- Always include clear labels and legends

---

## Tone of Voice

### Writing Style

- Clear and concise
- Neutral but friendly
- Action-oriented

### Example

❌ “Applicants rejected due to pipeline inefficiencies”  
✅ “429 applicants were not selected this period”

---

## Accessibility

- Ensure text contrast meets WCAG AA standards
- Click targets minimum 40px height
- Avoid relying solely on color to convey meaning

---

## Do & Don’t

### Do

- Use whitespace generously
- Highlight key metrics
- Keep interactions predictable

### Don’t

- Overuse bright colors
- Add unnecessary animations
- Overload dashboards with dense data

---

## Brand Summary

Hirezy’s brand reflects **clarity, trust, and modern hiring workflows**. Every design decision should reduce friction, surface insights, and make recruiters feel in control.
